تقویمِ محتوا لازم نیست پیچیده باشد. این ساختارِ ساده کافی است.

## ستون‌های پیشنهادی

1. **ستونِ آموزش:** حلِ یک مشکلِ مشخص.
2. **ستونِ روایت:** یک داستانِ کوتاه/واقعی.
3. **ستونِ موضع:** یک باورِ پرهزینه.
4. **ستونِ دعوت:** معرفیِ مسیر یا پیشنهاد.

## ریتم هفتگی

| روز | نوع |
| --- | --- |
| دوشنبه | آموزش |
| چهارشنبه | روایت |
| جمعه | موضع / دعوت |
