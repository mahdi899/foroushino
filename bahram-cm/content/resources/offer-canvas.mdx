این بوم کمک می‌کند پیشنهادت را شفاف کنی. هر بخش را با یک جمله پر کن.

## بخش‌های بوم

| بخش | پرسش راهنما |
| --- | --- |
| نتیجه | بعد از خرید، زندگیِ مخاطب چه فرقی می‌کند؟ |
| مخاطب | این دقیقاً برای کیست؟ |
| تمایز | چرا این، نه گزینه‌ی بعدی؟ |
| اثبات | چه چیزی باورپذیرش می‌کند؟ |
| قیمت | ارزش چطور توجیه می‌شود؟ |
| دعوت | قدم بعدی چیست؟ |

برای عمیق‌تر شدن، [شفافیتِ پیشنهاد](/insights/offer-clarity) را بخوان.
